# MLB Scouting & Evaluation

Scouting and evaluation in Minor League Baseball combines traditional subjective assessment with modern tracking and analytics. (source: [minor-league-baseball-ecosystem.md](../sources/minor-league-baseball-ecosystem.md))

## Scouting Roles

- **Area scouts:** Primary amateur acquisition; territory-based evaluation
- **Cross-checkers:** Calibration layer between area scouts and the scouting director; follow up on key players across territories (SABR)
- **Pro scouts / advance scouts:** Professional acquisition and valuation; includes international pro evaluation (e.g., Japan/Korea specialists)
- **Player development staff:** Internal evaluation for promotion and role decisions; year-round system with highest intensity at lower levels

## The 20–80 Scouting Scale

The industry-standard grading system:
- **50** = MLB average; **60** = above average; **70–80** = well above average
- **Position player tools:** Hit, Power, Run, Field, Throw
- **Pitcher evaluation:** Fastball, best secondary, other pitches, Command, Control
- Both **present** and **future** grades are assigned; "future overall" is key for prospect projection

## Key Tracked Measures

| Metric | Definition | Context |
|--------|-----------|---------|
| **Spin Rate** | RPM at release | Affects pitch trajectory and movement perception |
| **Exit Velocity** | Ball speed off the bat | Hard-hit ball ≥ 95 mph (Statcast) |
| **Launch Angle** | Vertical angle off the bat (degrees) | Key batted-ball descriptor |
| **Sprint Speed** | Statcast running speed framework | Elite thresholds defined |
| **Plate Discipline** | O-Swing%, Z-Swing%, Z-Contact%, K%, BB% | FanGraphs-defined; central for hitter approach and pitcher bat-missing profiles |

## Public Statcast Availability in Minors

The public Minor League Statcast dataset is **incomplete**:
- **Triple-A:** Complete tracking (2023; partial 2022)
- **Single-A:** Limited subset of parks (back to 2021)
- **Double-A:** Not included in the described public dataset
- **Defensive metrics** (e.g., Outs Above Average) not tracked via Baseball Savant for minors
- Organizations use significantly more internal technology than what is publicly available

## Level-by-Level Evaluation Emphasis

| Level | Key Evaluation Focus |
|-------|---------------------|
| **Rookie/Complex** | "Tools first" (bat speed, arm strength, athletic movement); readiness for instruction; raw stats are noisy |
| **Low-A** | Plate discipline (O-Swing/Z-Contact, BB/K); age-relative performance; contact quality where tracked |
| **High-A** | Skill consolidation; pitch mix stability and plan execution; role hypotheses (starter/reliever) |
| **Double-A** | Breaking ball recognition and chase control; command consistency; Rule 5 / 40-man timing evaluation |
| **Triple-A** | MLB role readiness (starter: 2–3 reliable pitches; reliever: elite bat-misser + strikes; hitter: offensive role + defensive fit); call-up planning |
